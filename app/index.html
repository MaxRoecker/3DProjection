Bem-vindo ao sistema de projeções.

Através do sistema é possível observar a projeção de qualquer objeto de duas maneiras: persectiva e paralela. Para definir qual projeção deseja, basta seleciona-lá logo ao lado do botão "Project" no topo da área definida para projeção.

Como definir o ponto de vista:

Para definir o ponto de vista, basta preenchermos os três campos no local solicitado, os campos são os vores x, y e z do ponto de vista.

Como definir um plano de pojeção:

Para definir um plano de projeção é necessário informar três pontos pertencentes ao mesmo. Os pontos são definidos nos campos logo abaixo ao label solicitante. Para cada ponto deve ser informada as coordenadas x, y e z.

Como definir um objeto para projetar:

Para projetar um objeto precisamos definir os vértices e suas superfícies:

  1. Vértices:
    • Definir as coordenadas x, y e z de cada vértice.
    • Para adicionar um novo vértice, basta clicar no botão com o ícone "+" logo abaixo.
  2. Superfícies:
    • Definir quais são os vértices que contemplam a superfície informada, seguindo a regra da mão direira.
    • Para adicionar uma nova superfície, basta clicar no botão com o ícone "+" logo abaixo.

Como escolher qual o tipo de projeção desejada:

As opções de projeção ficam ao lado do botão "Project", basta selecionar qual a desejada e clicar no botão para efetuar a projeção.

Como esconder as faces ocultas da projeção:

Para esconder as faces ocultas da projeção basta marca a opção XXXXXX ao lado do botão "Project".

Como mudar o ponto de vista:

Para mudar o ponto de vista existem duas opções:

  1. Definir um novo ponto de vista através nos campos x, y e z do ponto de vista
  2. Após uma porjeção realizada, apertar as teclas T1 T2 T3 T4... para deslocar o ponto de vista.
